如何在文章中引入图片

先说一下问题，在about 目录或者在这个一个层级新建一个目录，往里边放图片是可以生成的。但是在_post 里放图片，最后生成的目录里没有图片，只有一个html页面。 想改源码，但是找找没有找到。后来找了下既然官方都没有支持，那肯定也是不好改或者有它的想法，考虑到时间上的投入，就自己先按照自己的思路改下

1

首先，现在网上找了下，全是一样的不知道谁是原创的文章，要安装一个插件

npm install hexo-asset-image –save

不好使，卸载

npm uninstall hexo-asset-image

2

官网看看 https://hexo.io/zh-cn/docs/asset-folders

两种解决方式。 第一种很简单，在about相同层级新建images 目录，然后所有文章都引用里边的图片，缺点就是需要手动的向images 目录里放文件。

第二种使用标签插件，，，，，看都不想看。看完了发现是比较难受的方式，就是md 不支持的语法，必须要生成页面后才能看到效果

3

最后的解决办法，1 不用手动拷贝图片，2 就用md 的语法，这样md编辑器里也可以看到效果。

其实还是第一种解决办法，只是第一种的插件在hexo 3 里过期了，需用用另个一github上的

解决步骤：

1

修改配置文件_config.yml 为

post_asset_folder: true

2

我使用的是typora . 文件-> 偏好设置

3

npm install https://github.com/CodeFalling/hexo-asset-image --save

人生的一道道迷

发表于 2019-07-23 | 更新于 2019-08-02 | 分类于 日记 | 评论数：

我把自己的人生比作一道道的谜题。

不管是工作生活学习，我都遇到了各种难题。怎么去面对，去解决，这就是我的人生需要去做的事情

希望自己在清醒着的时候，记录下这些想法，当自己迷茫的时候能拉自己一把，让自己重新振作起来

有些问题，自己还没有找到答案，以后会持续更新

  1. 管不住自己，总是想玩。玩过之后又觉得对不起自几。 有时候定好的目标，但是稍微一点松懈，就放纵了。

    自律，真的不是一个容易得到的东西，身边的环境，自己的思想境界，身体的健康程度，对未来的预判，都决定了自己能不能坚持，持续地去做一件有意义的事情，以及这个事情在未来看确实有意义

  2. 如何看待生活中的不平等。比如

    1. 有让人提不起来干劲的：有的人当着你的面说你快写，写完了让我抄一下。你快做，做好了教教我怎么做。他们理智气壮，仿佛这就是你该做的。

      立地成佛，普度众生。如果自己做自己擅长的事情，还能给别人带来帮助，何乐而不为。但是前提确实是帮助了对方，这个判断就不是我自己的责任了。最终的结果好坏是大家的选择，我愿意做，你也要愿意承担结果

    2. 你可能奋斗了好几年赞一个首付，别人拆迁分了好几套

      虽然我有时候也跟自己说“做个俗人，贪财好色”， 但是，金钱、财富、荣誉、权势、地位等等都不过是身外之物，生不带来，死不带去。

    3. 父母辛辛苦苦工作才让自己上学，最后出来才证几千块，别人从小不缺钱花，也不上学，最后是你的老板，给你发工资，你替他干活。

      如果说活在当下的话， 当下我选择认命。还能怎么办，现在让我超过他们，不如死的痛快

  3. 怎么看待人际关系，有的人生来就善于交际，说话办事特别好使，而像我这样不会交际人，和一群这样的人在一起的时候感觉自己就是别人眼里的傻子。

    1 不必羡慕，也不要总是做自己不擅长的事情，知道自己不善于交际，就把时间花在自己擅长的事情上去。2 突破自己，不能完全活在自己的世界里，但是也不随波逐流，有选择的社交。

  4. 为什么定下的计划总是实现不了，想好的东西只是想想罢了

    坚持不下去，只是自己还没有达到那个水平.

  5. 梦想是永恒的，但时间不是。喜欢像小孩一样的活着，喜欢单纯美好的事情。但是生活却不是这样的

    就像一颗树，生命有春天也有冬天，最后总有枯萎的那一刻。这是一个过程，我也是人世间的一颗树，所以我也要经历这些，生老病死，我不能把时间定格在某一刻。我的思想是我身体的一部分，我的身体在一天天的变化，我的思想也是。所以今天的我不是昨天的我也不是下一刻的我。

  6. 为什么要活着，生活的意义到底是什么，这个问了一千遍的问题，总是不能给自己一个永恒的答案

    首先，在大多数情况下，并不是你活着，你就存在了。只有当大家数人数的时候，才会算你一个，在其他别的情况下，自己可能根本就没有存在的余地。如果真想要继续存在的时候，
